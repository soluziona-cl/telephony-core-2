# Service Domain

## Objetivo
Atención general, información, derivación y soporte.

## Reglas generales
- Respuestas informativas
- Derivación a especialistas cuando corresponde
- Soporte técnico y administrativo

## Bots

### default
Bot por defecto para atención general.

## Uso

El dominio se activa cuando el modo comienza con `voicebot_service_`:

```
voicebot_service_default → Bot por defecto
voicebot_service_soporte → Bot de soporte (futuro)
```

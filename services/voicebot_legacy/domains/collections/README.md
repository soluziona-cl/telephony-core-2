# Collections Domain

## Objetivo
Cobranza, compromisos y pagos.

## Reglas generales
- Respeto y empatía
- Negociación de pagos
- Recordatorios de compromisos

## Bots

### default
Bot por defecto para cobranza.

## Uso

El dominio se activa cuando el modo comienza con `voicebot_collections_`:

```
voicebot_collections_default → Bot por defecto
voicebot_collections_cobranza → Bot de cobranza (futuro)
```

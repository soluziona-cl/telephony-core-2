# Sales Domain

## Objetivo
Ventas, captación y campañas.

## Reglas generales
- Proactividad en ofertas
- Captación de leads
- Seguimiento de oportunidades

## Bots

### default
Bot por defecto para ventas.

## Uso

El dominio se activa cuando el modo comienza con `voicebot_sales_`:

```
voicebot_sales_default → Bot por defecto
voicebot_sales_ventas → Bot de ventas (futuro)
```
